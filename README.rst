==================
interaktiv.kyra
==================

KYRA integration for Plone.

Features
--------

- Installable Plone addon
- Browser layer support
- Ready for extension with KYRA functionality

Installation
------------

Install interaktiv.kyra by adding it to your buildout::

    [buildout]
    ...
    eggs =
        interaktiv.kyra

and then running ``bin/buildout``

Testing
-------

Run tests with::

    bin/test -s interaktiv.kyra
